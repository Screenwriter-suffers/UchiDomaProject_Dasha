# Проект: Интерактивный новостной портал

**Автор:** Нейморовец Дарья

### Что умеет сайт:
* **Динамические новости**: можно добавлять свои новости.
* **Категории**: удобная навигация по темам.
* **РЕКОМЕНДАЦИИ**: 
  * рекомендуется надеть наушники :)

### Как запустить:
1. Скопируй репозиторий.
2. Создай и активируй виртуальное окружение. `py -m venv .venv
source .venv/Scripts/activate`
3. напиши это в консоль: `pip install flask flask-sqlalchemy flask-wtf`
4. Запусти `python main.py`.
